## Installation

- `IOS`

  - `git clone`
  - `cd bitcicase`
  - `npm install`
  - `cd ios && pod install && cd ..`
  - `npx react-native run-ios`

- `Android`
  - `git clone`
  - `cd bitcicase`
  - `npm install`
  - `npx react-native run-android`
